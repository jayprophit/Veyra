SYSTEM OVERVIEW

WHAT THIS IS:
-------------
A customizable database framework that tracks financial wealth opportunities, 
credit systems, regulations, and business climates across different countries.

Start with UK as your baseline, then add countries/regions as you explore:
• Tax optimization opportunities
• Business expansion targets
• Property investment markets
• Crypto-friendly jurisdictions
• Manufacturing hubs
• Talent/immigration destinations

WHY YOU NEED THIS:
------------------
Your 12-year empire plan involves multiple pillars that can benefit from 
geographic diversification:
• Mechatronics manufacturing (cheaper labor, grants, shipping)
• Software/IP (tax-efficient jurisdictions, R&D credits)
• Property investment (higher yields, lower entry costs)
• Crypto (regulatory clarity, tax treatment)
• MMORPG (talent pools, server locations, player markets)

As your wealth grows, WHERE you operate becomes as important as WHAT you build.

TABLE 2: METRICS (What to Track)
=================================
The categories of data you want to compare across countries.

Fields:
-------
• Metric Name (Primary)      → Personal Credit Score, CGT Rate
• Category                    → Personal Finance / Tax / Business / Crypto / Real Estate / Immigration / Employment
• Sub-Category                → Capital Gains / Income Tax / Corporate / Property
• Data Type                   → Number / Text / Currency / Percentage / Date / Boolean / URL
• Unit                        → % / £ / $ / € / Years / Months
• Description                 → What this metric measures
• Importance Weight           → 1-10 (how much this matters to your goals)
• UK Baseline Value           → Current UK value for comparison
• Data Source                 → URL or source name
• Update Frequency            → Daily / Weekly / Monthly / Quarterly / Annually / As Needed
• Last Updated                → Date field
• Formula/Calculation         → How to calculate (if applicable)

⚠️ TAX RESIDENCY COMPLEXITY:
-----------------------------
• Moving countries doesn't automatically change tax residency
• UK has statutory residence test (SRT) - 183 days rule
• You may remain UK tax resident even living abroad
• Seek professional tax advice before relocating

⚠️ CRYPTO TAX NUANCES:
-----------------------
• "0% tax after 1 year" often has conditions
• May still need to report even if no tax due
• Tax treaties between countries complicate things
• Professional crypto tax accountant essential

• Digital nomad visas have income requirements
• Citizenship by investment programs change frequently
• Always check current rules before planning
