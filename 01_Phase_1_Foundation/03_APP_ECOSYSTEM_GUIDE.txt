1. NET WORTH & FINANCIAL TRACKING
================================================================================

PRIMARY: GOOGLE SHEETS (FREE)
-------------------------------
Why: Cloud-based, auto-save, real-time collaboration, accessible anywhere
Best For: Daily tracking, simple formulas, sharing with advisors

SETUP:
1. Create Google account (if not done)
2. Go to sheets.google.com
3. Create "Financial Master Tracker"
4. Enable offline mode (File → Settings)

TRACKERS TO BUILD:
------------------
[ ] NET WORTH DASHBOARD
  Columns: Date | Assets | Liabilities | Net Worth | Change | Notes
  Formulas: =SUM(B2:F2)-SUM(G2:J2) for net worth
  Charts: Line graph showing net worth over time
  Update: Monthly (1st of month)

[ ] 12-YEAR PROJECTION
  Rows: Years 1-12
  Columns: Age | Software Revenue | Hardware Value | Property | MMORPG | Total
  Formulas: Cumulative growth calculations
  Charts: Stacked area chart showing all pillars

[ ] MONTHLY BUDGET
  Rows: Income categories | Expense categories
  Columns: Budgeted | Actual | Variance
  Formulas: =Actual-Budgeted for variance
  Conditional formatting: Red if overspending

[ ] DEBT SNOWBALL TRACKER
  Rows: Resolve, Terry, HMV, Jason
  Columns: Balance | Min Payment | Extra Payment | Remaining | Payoff Date
  Formulas: =TODAY()+Remaining/Extra for payoff date
  Charts: Bar chart showing debt reduction

[ ] EMERGENCY FUND PROGRESS
  Target: £6,087
  Current: =SUM(Pot transfers)
  Progress %: =Current/Target
  Charts: Progress bar with milestone markers (£1k, £3k, £6k)

TEMPLATES AVAILABLE:
--------------------
• Tiller Money (til.com) - Automated import, £6/month
• Vertex42 (vertex42.com) - Free Excel/Google Sheets templates
• Spreadsheet Point - Free budget templates


ALTERNATIVE: MICROSOFT EXCEL (£6/month with 365)
------------------------------------------------
Why: More powerful calculations, better for complex modeling, industry standard
Best For: Quarterly deep dives, advanced financial modeling, heavy data analysis

ADVANTAGE OVER GOOGLE SHEETS:
• More advanced formulas (INDEX/MATCH, array formulas)
• Better for large datasets (>10,000 rows)
• Power Query for data transformation
• PivotTables more powerful
• VBA macros for automation

SETUP IF USING EXCEL:
---------------------
1. Microsoft 365 Personal: £6/month (includes Excel, Word, PowerPoint, OneDrive)

3. Save to OneDrive for cloud sync
4. Enable auto-save

RECOMMENDED: Use BOTH

FEATURES TO USE:
----------------

   - Verify auto-categorizations are correct
   - Fix any "missing cost basis" warnings
   - Tag transfers between your own wallets

[ ] TAX REPORT GENERATOR
   - Select: UK, HMRC
   - Tax year: 6 April - 5 April
   - Download: SA108 Capital Gains Summary

[ ] TAX SINKING FUND TRACKER
   - Link to Monzo Tax Pot

12. SECURITY BEST PRACTICES
================================================================================

ACCOUNT SECURITY:
-----------------
• Use strong, unique passwords for each app
• Enable 2FA on EVERY account (Google Authenticator)
• Use password manager (Bitwarden FREE or 1Password £3/month)
• Never share API keys (especially with withdrawal permissions)
• Review connected apps monthly (remove unused)

DATA BACKUP:
------------
• Google Sheets: Auto-saves to Google Drive
• Excel: Save to OneDrive (auto-sync)
• Airtable: Weekly CSV exports
• QuickBooks: Monthly PDF reports
• Koinly: Annual tax report downloads
• Use 3-2-1 backup rule: 3 copies, 2 different media, 1 offsite

PRIVACY:
--------
• Review privacy policies (especially for financial apps)
• Limit data sharing where possible
• Use read-only API connections (never enable withdrawals)
• Consider VPN for sensitive financial work
