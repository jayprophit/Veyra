-------------
1. User opens User_Profile.xlsx

3. Selects employment status from dropdown
4. Enters age, income, expenses

KEY FORMULAS FOR DYNAMIC LOOKUP:
---------------------------------

1. GET CGT RATE FOR USER'S COUNTRY:
```excel
=VLOOKUP('00_User_Profile'!B2, Global_Tax_Database!A:Z, 
         MATCH("CGT_Rate_Basic", Global_Tax_Database!1:1, 0), FALSE)
```
Returns 14% for UK, 28% for Portugal, 0% for Belgium, etc.

2. GET ISA/TAX-FREE WRAPPER ALLOWANCE:
```excel
=IF(ISNUMBER(VLOOKUP('00_User_Profile'!B2, Global_Tax_Database!A:Z, 
         MATCH("ISA_Allowance", Global_Tax_Database!1:1, 0), FALSE)), 
    VLOOKUP('00_User_Profile'!B2, Global_Tax_Database!A:Z, 
         MATCH("ISA_Allowance", Global_Tax_Database!1:1, 0), FALSE), 
    "No tax-free wrapper available")
```
Returns £20,000 for UK, "No tax-free wrapper" for Portugal.

3. GET PENSION ACCESS AGE:
```excel
=VLOOKUP('00_User_Profile'!B2, Global_Tax_Database!A:Z, 
         MATCH("Pension_Access_Age", Global_Tax_Database!1:1, 0), FALSE)
```
Returns 57 for UK, 66 for Portugal, 59½ for US.

4. CHECK IF DIGITAL NOMAD VISA AVAILABLE:
```excel
=IF(VLOOKUP('00_User_Profile'!B2, Global_Tax_Database!A:Z, 
         MATCH("Digital_Nomad_Visa", Global_Tax_Database!1:1, 0), FALSE)="Yes", 
    "Digital nomad visa available", "No digital nomad visa")
```

EXAMPLE SCENARIO: UK Freelancer with US Client
-----------------------------------------------

USER PROFILE:
- Residence: United Kingdom
- Employment: Contract/Freelance
- Client Location: United States
- Invoice Amount: $10,000 USD

DYNAMIC CALCULATION:

| Line Item | Formula / Lookup | Result |
|-----------|------------------|--------|
| Gross Income (USD) | User enters | $10,000 |
| Exchange Rate (USD to GBP) | =GOOGLEFINANCE("CURRENCY:USDGBP") | 0.78 |
| Convert to GBP | =B2 * B3 | £7,800 |
| US Withholding Tax Rate | =VLOOKUP("United States", Tax_Treaty_Matrix, MATCH("UK", Headers, 0), FALSE) | 0% (treaty exempts services) |
| UK Income Tax Rate | =VLOOKUP("United Kingdom", Global_Tax_Database, MATCH("Income_Tax_Basic", Headers, 0), FALSE) | 20% |
| UK Income Tax Due | =B4 * B7 | £1,560 |
| UK NICs Class 2 Flat Rate | £179.40/year | £179.40 |
| UK NICs Class 4 (9% over £12,570) | =IF(B4>12570, (B4-12570)*0.09, 0) | £0 (income below threshold) |
| Total UK Tax/NICs | =SUM(B9:B11) | £1,739.40 |
| Net After Tax | =B4 - B12 | £6,060.60 |
| Effective Tax Rate | =B12/B4 | 22.3% |

CHANGE COUNTRY TO PORTUGAL - AUTO-UPDATE:
------------------------------------------
| Line Item | Formula | Result |
|-----------|---------|--------|
| Gross Income (EUR) | €10,000 | €10,000 |
| Exchange Rate (USD to EUR) | 0.92 | 0.92 |
| Portuguese Income Tax | Lookup 14.5-48% scale | €1,450 (14.5% on first €10,000) |
| Portuguese Social Security | 21.4% (if sole trader) | €2,140 |
| NHR Regime (if applicable) | 0% for foreign-sourced | €0 |
| Net After Tax | | €6,410 |

Same invoice, different country = entirely different tax outcome. All automatic.

1. VLOOKUP COUNTRY PARAMETER:
```excel
=VLOOKUP($User_Country, Global_Tax_Database!$A:$Z, 
         MATCH($Parameter_Name, Global_Tax_Database!$1:$1, 0), FALSE)
```

2. VLOOKUP EMPLOYMENT PARAMETER:
```excel
=VLOOKUP($Employment_Status, Employment_Config!$A:$Z, 
         MATCH($Parameter_Name, Employment_Config!$1:$1, 0), FALSE)
```

3. INDEX-MATCH ALLOCATION:
```excel
=INDEX(Allocation_Matrix!$C:$G, 
       MATCH(1, (Allocation_Matrix!$A:$A=$Risk_Profile) * 
                 (Allocation_Matrix!$B:$B=$Employment_Status), 0), 
       COLUMN()-2)
```

4. LIVE CURRENCY CONVERSION:
```excel
=Amount * GOOGLEFINANCE("CURRENCY:" & Source_Currency & Target_Currency)
```

5. CONDITIONAL SECTION DISPLAY:
```excel
=IF(ISNUMBER(VLOOKUP($Country, Database, MATCH($Param, Headers, 0), FALSE)), 
    "SHOW SECTION", "HIDE SECTION")
```

6. EMERGENCY FUND CALCULATION:
```excel
=Monthly_Expenses * VLOOKUP($Employment, Employment_Config, 
                            MATCH("Emergency_Months", Headers, 0), FALSE)
```

7. INVESTMENT LIMIT CALCULATION:
```excel
=Monthly_Income * VLOOKUP($Employment, Employment_Config, 
                          MATCH("Investment_Limit_Pct", Headers, 0), FALSE)
```

8. TAX CALCULATION (CROSS-BORDER):
```excel
=Gross_Income * VLOOKUP($Country, Tax_Database, 
                        MATCH("Income_Tax_Rate", Headers, 0), FALSE)
```
