FILE NAMING CONVENTIONS
================================================================================

Use this format for ALL files: YYYY-MM-DD_Descriptive_Name_vX.X.ext

COMPONENTS:
-----------
YYYY-MM-DD     = Date (year-month-day) - use creation or effective date
Descriptive_Name = Clear, searchable name (use underscores for spaces)
vX.X           = Version number (v1.0 = initial, v1.1 = minor edit, v2.0 = major revision)
ext            = File extension (.xlsx, .docx, .pdf, .txt)

EXAMPLES:
---------
2026-04-21_Net_Worth_Snapshot_v1.0.xlsx
2026-01-31_Self_Assessment_SA100_v1.0.pdf

Crypto_Portfolio_Snapshot_2026-04-21_v1.0.xlsx
Debt_Snowball_Plan_2026_v1.2.xlsx

VERSION CONTROL RULES:
-----------------------
вҖў v1.0 = First complete version
вҖў v1.1 = Minor correction/typo fix
вҖў v1.2 = Additional data added (e.g., new month)
вҖў v2.0 = Major restructure or methodology change
вҖў Always increment when making changes
вҖў Keep 2-3 previous versions, then archive

SEARCH TIPS:
------------
вҖў Use consistent keywords: "Statement", "Tracker", "Report", "Summary"
вҖў Always include year: "2026", "2025"
