в–Ў ITEM 1: IDENTIFY FILE TYPE

  в–Ў Working Data/Tracking (goes to 07_Working_Files\)
  в–Ў Reference Material (goes to 06_Reference\)
  в–Ў Technical Guide DOCX (goes to 08_System_Guides\)
  в–Ў Replacing Existing File (requires archiving)

в–Ў ITEM 2: DETERMINE CORRECT FOLDER
  Based on file type above:
  
  If Strategy/Plan:
    в–Ў Months 1-12 вҶ’ 01_Phase_1_Foundation\
    в–Ў Years 2-3 вҶ’ 02_Phase_2_Launch\
    в–Ў Years 4-5 вҶ’ 03_Phase_3_Expansion\
    в–Ў Years 6-7 вҶ’ 04_Phase_4_Scaling\
    в–Ў Years 8+ вҶ’ 05_Phase_5_Empire\
    в–Ў Quick Reference вҶ’ 06_Reference\
  
  If Working Data:
    в–Ў Net worth tracking вҶ’ 07_Working_Files\01_Net_Worth\
    в–Ў Income/expenses вҶ’ 07_Working_Files\02_Income_Expenses\
    в–Ў Investment records вҶ’ 07_Working_Files\03_Investments\

    в–Ў Trading logs вҶ’ 07_Working_Files\06_Trading_Logs\
    в–Ў CAD/designs вҶ’ 07_Working_Files\07_Business_Designs\
  
  If Technical Guide (DOCX):
    в–Ў 08_System_Guides\

в–Ў ITEM 3: CHECK NAMING CONVENTION
  Does your filename follow the rules?
  
  For Phase folders:
    Format: [XX]_[TOPIC]_[TYPE].txt
    Example: 04_PROPERTY_INVESTMENT_GUIDE.txt
    вң“ Two-digit number
    вң“ Descriptive topic
    вң“ Document type
    вң“ NO spaces (use underscores)
    вң“ NO version numbers (v2, final, etc.)
  
  For Working Files:
    Format: YYYYMMDD_[Description].[ext]
    Example: 20260421_April_Net_Worth.xlsx
    вң“ Date first (8 digits)
    вң“ Descriptive name
    вң“ Appropriate extension
  
  For System Guides:
    Format: [XX]_[TOPIC]_[SUBTOPIC].docx
    Example: 19_ADVANCED_TAX_STRATEGIES.docx
    вң“ Two-digit number
    вң“ Topic and subtopic

в–Ў ITEM 4: VERIFY NO DUPLICATES
  в–Ў Check target folder for existing file with same name
  в–Ў If updating existing file:
    вҶ’ Archive old version first (see ITEM 5)
    вҶ’ Keep same filename for new version
  в–Ў If different content but similar name:
    вҶ’ Use next available number

в–Ў ITEM 5: ARCHIVE IF REPLACING (Skip if new file)
  If this replaces an existing file:
  
  в–Ў Open old file location
  в–Ў Copy old file to: 09_Archive\Consolidated_Files\
  в–Ў Rename archived file: OLD_YYYYMMDD_[original_name].[ext]
    Example: OLD_20260421_01_SIMPLIFIED_CHRONOLOGICAL_GUIDE.txt
  в–Ў Verify old file is safely archived
  в–Ў Now save new file to original location with original name

  в–Ў Check Documents folder
  в–Ў Delete any duplicate/temporary copies
  в–Ў Only keep file in Financial Master location
