================================================================================
THE GOLDEN RULES
================================================================================

RULE 1: EVERY FILE HAS A HOME
-------------------------------
Before saving ANY file, ask: "Which folder does this belong in?"
Use this guide to determine the correct location.

RULE 2: FOLLOW THE NAMING CONVENTION
-------------------------------------
All files MUST use the approved naming format.
Inconsistent naming = chaos.

RULE 3: NO FILES IN ROOT DIRECTORY
-----------------------------------
The main "Financial Master" folder should ONLY contain:
- 10 phase folders (00-09)
- README.txt
- This maintenance protocol

NO exceptions. No loose files ever.

RULE 4: ARCHIVE IMMEDIATELY
----------------------------
When a file is superseded, move it to 09_Archive\ within 24 hours.
Don't let obsolete files accumulate.

RULE 5: WORKING FILES ONLY IN 07_Working_Files\
------------------------------------------------
Active data (spreadsheets, receipts, logs) goes ONLY in 07_Working_Files\ subfolders.
Reference guides go in phase folders.

FILE TYPE GUIDELINES

-----------------

PDFS:

------------------------

================================================================================
ADDING NEW FILES - STEP-BY-STEP PROCESS
================================================================================

STEP 1: IDENTIFY THE FILE TYPE
--------------------------------
Is it:
  □ Strategy/Plan → Go to Step 2
  □ Working data/Active tracking → Go to Step 3
  □ Reference material → Go to Step 4
  □ Technical guide → Go to Step 5
  □ Replacing existing file → Go to Step 6

STEP 2: STRATEGY/PLAN FILE
--------------------------
1. Determine which phase it belongs to:
   • Months 1-12 → 01_Phase_1_Foundation\
   • Years 2-3 → 02_Phase_2_Launch\
   • Years 4-5 → 03_Phase_3_Expansion\
   • Years 6-7 → 04_Phase_4_Scaling\
   • Years 8+ → 05_Phase_5_Empire\

2. Check existing files in that folder
   Determine the next available number

3. Name the file: [XX]_[TOPIC]_[TYPE].txt
   Example: 04_PROPERTY_INVESTMENT_GUIDE.txt

4. Save to the phase folder

5. Update 00_START_HERE\03_COMPLETE_MASTER_INDEX.txt
   Add entry for the new file

STEP 3: WORKING DATA FILE
-------------------------
1. Determine the appropriate subfolder:
   • Financial tracking → 07_Working_Files\01_Net_Worth\
   • Income/expenses → 07_Working_Files\02_Income_Expenses\
   • Investment records → 07_Working_Files\03_Investments\
   • Tax documents → 07_Working_Files\04_Tax_Records\
   • Legal docs → 07_Working_Files\05_Legal_Documents\
   • Trading data → 07_Working_Files\06_Trading_Logs\
   • CAD/designs → 07_Working_Files\07_Business_Designs\

2. Name the file: YYYYMMDD_[Description].[ext]
   Example: 20260421_April_Net_Worth.xlsx

3. Save to the appropriate subfolder

4. No index update needed (working files self-organize by date)

STEP 4: REFERENCE MATERIAL
--------------------------
1. Save to: 06_Reference\

2. Name the file: [XX]_[TYPE]_REFERENCE.txt
   Example: 02_TAX_CHEAT_SHEET.txt

3. Update 00_START_HERE\03_COMPLETE_MASTER_INDEX.txt

STEP 5: TECHNICAL GUIDE (DOCX)
------------------------------
1. Save to: 08_System_Guides\

2. Determine next available number (check existing files)

3. Name the file: [XX]_[TOPIC]_[SUBTOPIC].docx
   Example: 19_ADVANCED_TAX_STRATEGIES.docx

4. No index update needed (System Guides self-contained)

STEP 6: REPLACING/UPDATING EXISTING FILE
-----------------------------------------
1. If file is a MAJOR revision (new version):
   a. Move OLD version to 09_Archive\Consolidated_Files\
   b. Add date stamp to archived filename:
      Example: OLD_20260421_01_SIMPLIFIED_CHRONOLOGICAL_GUIDE.txt
   c. Save NEW version with same name in original location

2. If file is a MINOR update (same version, corrections):
   a. Overwrite existing file
   b. No archiving needed

3. Update 00_START_HERE\03_COMPLETE_MASTER_INDEX.txt if:
   • File name changed
   • File location changed
   • File description needs updating

================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "I don't know which phase folder to use"
SOLUTION: Check your current timeline. What year/month are you in?
  • If Months 1-12 → 01_Phase_1_Foundation\
  • If Year 2-3 → 02_Phase_2_Launch\
  • etc.

PROBLEM: "File name already exists"
SOLUTION: 
  1. Check if you're updating existing file → Archive old, keep name
  2. Check if different content → Use next available number

PROBLEM: "Folder is getting too cluttered"
SOLUTION:
  1. Archive old/superseded files
  2. Create subfolders within phase if needed (rare)
  3. Consider if files belong elsewhere

PROBLEM: "I saved to wrong folder"
SOLUTION:
  1. Move to correct folder immediately
  2. Don't leave duplicates
  3. Update any links/references

PROBLEM: "Can't find a file I saved"
SOLUTION:
  1. Check 00_START_HERE\03_COMPLETE_MASTER_INDEX.txt
  2. Use search by date or keyword
  3. Check Archive (might have been moved)
