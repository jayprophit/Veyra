5.1 OPEN-SOURCE FRAMEWORKS
===========================

A. TRADINGAGENTS (Multi-Agent Trading)
---------------------------------------
GitHub: github.com/tradingagents/trading-agents
Cost: Free (open source)
Autonomy Level: 4-5 (customizable)

Architecture:
```
TradingAgents Framework:
├── Fundamental Analyst Agent
│   └── Analyzes company financials, earnings, ratios
├── Sentiment Analyst Agent
│   └── Processes news, social media, on-chain data
├── Technical Analyst Agent
│   └── Chart patterns, indicators, price action
├── Risk Manager Agent
│   └── Position sizing, stop-losses, portfolio heat
├── Trader Agent
│   └── Executes trades based on consensus
└── Orchestrator
    └── Coordinates agents, resolves conflicts
```

Setup:
```python
# Example: Custom BTC Trading Agent
from tradingagents import Agent, Orchestrator

# Define agents
fundamental = Agent(type='fundamental', focus=['BTC', 'macro'])
sentiment = Agent(type='sentiment', sources=['twitter', 'reddit', 'news'])
technical = Agent(type='technical', indicators=['rsi', 'macd', 'bollinger'])
risk = Agent(type='risk', max_position=0.05, max_drawdown=0.10)
trader = Agent(type='trader', exchange='binance', test_mode=False)

# Create orchestrator with your rules
orch = Orchestrator(
    agents=[fundamental, sentiment, technical, risk, trader],
    rules={
        'never_sell': ['BTC_HODL_WALLET'],
        'max_daily_trades': 5,
        'stop_loss': 0.03,
        'take_profit': 0.10,
        'consensus_threshold': 0.7  # 70% of agents must agree
    }
)

# Run autonomously
orch.run_autonomous_mode(interval='1h')
```
