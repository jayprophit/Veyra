[STEP 2: GOOGLE SHEETS IMPORT]
1. Open sheets.google.com and sign in
2. Create new spreadsheet: "Financial_Command_Center_v1"
3. For each CSV file:
   a. File → Import
   b. Upload the CSV file
   c. Choose: "Replace spreadsheet" for first, "Insert new sheet(s)" for rest
   d. Import location: Replace data at selected cell (A1)
   e. Separator type: Comma
   f. Click "Import data"

[STEP 3: EXCEL IMPORT]
1. Open Excel and create new workbook: "Financial_Command_Center.xlsx"
2. For each CSV:
   a. Data → Get Data → From File → From Text/CSV
   b. Select the CSV file
   c. File Origin: 65001: Unicode (UTF-8)
   d. Delimiter: Comma
   e. Data Type Detection: Based on first 200 rows
   f. Load to: New sheet

[PRICE PREDICTION MODEL]
Model Type: Gradient Boosting Ensemble
Features Used:
→ Returns (daily percentage change)
→ Moving averages (7-day, 30-day)
→ Volatility (7-day, 30-day standard deviation)
→ Momentum (7-day, 30-day price change)
→ RSI (Relative Strength Index)
→ Bollinger Band position
→ Volume ratios

Forecast Horizon: 7 days
Retrain Frequency: Every 30 days
Accuracy Target: >65% directional accuracy

[RISK ASSESSMENT MODEL]
Metrics Calculated:
→ Value at Risk (VaR) at 95% confidence
→ Conditional VaR (Expected Shortfall)
→ Sharpe Ratio (risk-adjusted returns)
→ Maximum Drawdown (worst peak-to-trough)
→ Volatility (annualized standard deviation)
→ Skewness (return distribution asymmetry)
→ Kurtosis (tail risk measure)

[TRADING SIGNAL GENERATION]
Signal Types:
→ BUY (allocation below target, positive prediction)
→ SELL (allocation above target, negative prediction)
→ HOLD (within threshold or low confidence)

Confidence Thresholds:
→ Strong: >80% confidence
→ Moderate: 65-80% confidence
→ Weak: <65% (ignored)

Risk Checks:
→ Daily loss limit: 2% max
→ VaR limit: 5% max
→ Allocation limit: 20% max per asset

[ADDING NEW ASSETS]
1. Add to 01_Platform_Inventory_Master.csv
2. Create new row in Dashboard for allocation
3. Add to 00_Master_System_Config.json assets section
4. Restart AI engine

[CHANGING RISK TOLERANCE]
Edit 00_Master_System_Config.json:
{
  "automation_settings": {
    "risk_tolerance": "high",  // low, medium, high
    "max_daily_loss_pct": 3.0,
    "rebalance_threshold": 0.10
  }
}

[ENABLING LIVE TRADING]
⚠️ WARNING: Only enable after 6+ months of simulation

Edit 00_Master_System_Config.json:
{
  "automation_settings": {
    "enable_live_trading": true,
    "simulation_mode": false
  }
}

Also configure API keys for your exchanges in the api_keys section.

PART 7: TROUBLESHOOTING
================================================================================

[SPREADSHEET ISSUES]
Problem: Formulas show #REF! error
Solution: Check that sheet references exist (Outgoings_Detail, etc.)

Problem: Circular reference warning
Solution: Ensure B8 doesn't reference itself. Check formula chain.

Problem: Percentages display wrong
Solution: Format cells as Percentage (not Number)

[AI ENGINE ISSUES]
Problem: Module not found errors
Solution: pip install pandas numpy scikit-learn requests schedule

Problem: API rate limit errors
Solution: The engine has built-in rate limiting. Just wait 1 minute.

Problem: Price data shows None
Solution: Check internet connection. Some APIs require keys.

Problem: Config file not found
Solution: Create 00_Master_System_Config.json from template

[INTEGRATION ISSUES]
Problem: CSV export fails
Solution: Check write permissions in the folder

Problem: Live prices not updating in spreadsheet
Solution: Re-import the CSV. Google Sheets doesn't auto-refresh.
