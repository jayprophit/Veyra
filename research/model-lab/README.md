# Model Lab

Research workstream for model evaluation, retrieval, fine-tuning, and eventual proprietary-model decisions.

## Sequence

1. local inference with existing open models
2. retrieval and tool-use evaluations
3. specialist fine-tunes
4. small domain models or adapters
5. from-scratch training only when justified by data, compute, and benchmarks

## Not A Current Product Claim

No from-scratch Veyra LLM exists in the active runtime today. The model lab exists so that decision can be made with evidence instead of assumption.
