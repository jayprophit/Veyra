# Desktop Application

## Overview

Electron-based desktop application for Veyra with native OS integration, advanced charting, and keyboard shortcuts.

## Technology Stack

- Electron
- React
- TypeScript
- Node.js
- Native OS APIs

## Features

- Native OS integration
- Advanced charting
- Keyboard shortcuts
- Multi-monitor support
- Offline capability
- System tray integration
- File system access
- Native notifications

## Development

```bash
cd apps/desktop
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Platform Support

- Windows
- macOS
- Linux
