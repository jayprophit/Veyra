# Tablet Application

## Overview

React Native tablet application for Veyra with tablet-optimized UI, touch gestures, and landscape/portrait support.

## Technology Stack

- React Native
- TypeScript
- Expo
- React Navigation
- Gesture Handler

## Features

- Tablet-optimized UI
- Touch gestures
- Landscape/portrait support
- Split-screen support
- Stylus support
- Offline capability
- Push notifications
- Biometric authentication

## Development

```bash
cd apps/tablet
npm install
npm run dev
```

## Platform Support

- iPad (iOS)
- Android Tablets
- Windows Tablets
