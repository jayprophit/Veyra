# Smart TV Application

## Overview

React Native TV application for Veyra with TV-optimized UI, remote control support, and voice commands.

## Technology Stack

- React Native TV
- TypeScript
- Expo
- React Navigation
- Voice Commands

## Features

- TV-optimized UI
- Remote control support
- Voice commands
- 10-foot UI
- Large text and buttons
- Landscape-only support
- Offline capability
- Push notifications

## Development

```bash
cd apps/smart-tv
npm install
npm run dev
```

## Platform Support

- Android TV
- Apple TV
- Samsung Tizen
- LG webOS
