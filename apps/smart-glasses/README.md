# Smart Glasses Application

## Overview

React Native application for Veyra with AR capabilities, voice commands, and head-up display.

## Technology Stack

- React Native
- TypeScript
- Expo
- AR Framework
- Voice Commands
- Head-Up Display

## Features

- AR capabilities
- Voice commands
- Head-up display
- Gesture controls
- Notifications
- Offline capability
- Battery optimization
- Low-latency display

## Development

```bash
cd apps/smart-glasses
npm install
npm run dev
```

## Platform Support

- Apple Vision Pro
- Meta Quest
- XREAL Air
- Ray-Ban Meta
